# Rail Rhythm 扩展

## 交互窗口

运行 `label.py ` 可以使用交互窗口模式使用工具，功能基本相同。

## 绘图

要实现诸如[样例视频](https://www.bilibili.com/video/BV1sWRNYNEg7)所示的效果，可以如下操作：

1. 在项目文件夹下，建立一个pictures文件夹，其中再建立一个source子文件夹，里面放一张背景图片（标准分辨率为3860*2140），命名为background.png。
2. 运行 `label.py ` ，导入某天数据之后，输入`rank_station 动分动集普速`，程序会自动开始对全国车站的三类车次进行统计，依次生成车站的信息卡。生成数量足够之后掐断即可。如果只统计三类中的一部分那么就依次对输入进行修改。不能统计市域，因为无法从12306中获得一些市域的信息（如杭绍甬市域）。
3. 图片简单剪辑成视频。
