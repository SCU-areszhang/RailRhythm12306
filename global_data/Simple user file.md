# Rail Rhythm 简明指南

* 所有输入都不区分大小写

### 设置日期

先更改日期设置。按`2025-02-23`的格式输入您想要查询的日期。如果存在对应日期的数据包，程序会自动读取。

### 联网导入

不存在对应日期的数据时，输入`import A`，在线爬取12306网站的数据，稍等一分钟即可。只想要某个字头的数据，输入`import GDC`形式的命令。

提高爬取成功率，在原有命令基础上后加空格和数字`1`或`2`，但速度依次变慢。

如果需要保存导入的数据，可以输入命令`Save`。

### 查车次

直接输入`.`或者`。`，然后紧跟车次号，如`.G101`。

### 查车站
输入带“站”字的站点中文名，如`杭州东站`。 

若要对车次方向进行筛选，在原有命令基础上，后加`+`输入筛选关键字，可以叠加。如`杭州东站+stup`。

* `st`始发 `ed`终到 `ps`中间经停
* `dn`下行 `up`上行

### 查站站

将起止站名用`-`连接，多个选择用`/`分隔，且都不带“站”字和空格，如`北京/北京南-上海/上海虹桥/上海南`。

也可以用`--`连接两个城市的名称，如`北京--上海`。只支持地级以上城市。

可以改变结果的排序方式，加`+`后输入关键字，如`北京--上海+v`。

* `st`按出发时间从早到晚，默认
* `ed`按到达时间从早到晚
* `v`按耗时从短到长

### 类型筛选

在原有命令基础上，后加`*`输入字头（纯数字用`P`表示），如`杭州东站+stup*GD`、`北京/北京南-上海/上海虹桥/上海南+ed*ZTKP`、`北京--上海+v*G`

### 跳转

查询结果中每行最前面的数字是索引，可直接输入跳转。

### 回溯

输入`<<`或`《《`回退，输入`>>`或`》》`前进。 

### 补充说明

有些用户可能会把本时刻表查询工具和其他工具的结果作对照，而结果往往不完全一致。这有可能是因为某一边的数据未更新，请以实际为准。
